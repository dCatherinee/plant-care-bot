# Telegram-бот «Уход за комнатными растениями» (MVP)

Пет-проект: Telegram-бот, который помогает вести список домашних растений, отмечать уход (полив/удобрение) и получать напоминания по заданным интервалам.

---

## Базовые требования
- Go версии 1.25.7

## Команды
- make test - запуск тестов
- make lint - запуст форматирования и анализатора


## Возможности (MVP)

### Главное
- `/start` + главное меню на кнопках
- Раздел **Помощь/инструкция**

### Растения
- ➕ Добавить растение
- 📋 Показать список растений
- 🗑️ Удалить растение

### Уход
- ✅ Отметить **полив**
- ✅ Отметить **удобрение**
- ↩️ **Undo** — отменить последнюю отметку (полив/удобрение), если нажали случайно

### Напоминания
- 🔔 Уведомления о поливе/удобрении на основе интервалов ухода
- 🕒 Настройка времени уведомлений (в пределах окна уведомлений)

### Настройки
- 🌍 Таймзона
- 🪟 Окно уведомлений (например, `10:00–21:00`)
- 🔕 Включить/выключить уведомления

---

## Сценарий использования

1. Нажмите `/start` и откройте меню.
2. Добавьте растения (название + интервалы полива/удобрения).
3. Отмечайте уход кнопками «Полив» / «Удобрение».
4. Если отметка была случайной — нажмите **Undo**.
5. Включите напоминания и задайте таймзону + окно уведомлений.

## Логика напоминаний (кратко)

- Для каждого растения хранятся:
  - последний полив и интервал полива
  - последняя подкормка и интервал подкормки
- Бот отправляет уведомление, когда наступает “дедлайн” по интервалу.
- Уведомления отправляются только:
  - если они включены в настройках
  - в пределах окна уведомлений пользователя
  - с учётом таймзоны пользователя

---

## Структура данных (идея)

Минимально потребуется хранить:
- `users`: id пользователя, таймзона, окно уведомлений, включены ли уведомления
- `plants`: id, user_id, name, интервалы (полив/удобрение), даты последних отметок
- `care_logs`: история отметок (тип действия, время) — нужна для Undo
